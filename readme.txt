Месячный скринер моментума для российского рынка акций с бэктестингом и рекомендациями.

== Описание ==

Momentum Screener — это WordPress плагин для анализа и скрининга российских акций на основе стратегии моментума.

**Основные возможности:**

* Расчёт momentum для российских акций
* Настраиваемые параметры стратегии
* Полный бэктестинг с метриками
* Текущие рекомендации для покупки
* Визуализация результатов (графики)
* Поддержка дивидендов
* Фильтр волатильности
* Риск-корректированный momentum
* Фильтр по диапазону доходности
* Фильтрация по произвольному списку тикеров (для нескольких скринеров на разных страницах)

**Метрики:**

* Общая и годовая доходность
* Коэффициент Шарпа
* Коэффициент Сортино
* Максимальная просадка
* Волатильность портфеля

== Установка ==

1. Загрузите папку плагина в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню «Плагины» в WordPress
3. Перейдите в Настройки > Momentum Monthly
4. Загрузите Excel-файл с данными о ценах акций
5. Добавьте шорткод `[momentum_monthly]` на любую страницу

== Настройка данных ==

Для работы плагина необходим Excel-файл (.xlsx) со следующей структурой:

1. Лист с названием «цены»
2. Первый столбец: Time (даты)
3. Остальные столбцы: тикеры акций с ценами (каждая строка — один месяц)
4. Опционально: лист «дивиденды» или «Дивид» в том же формате

== Использование ==

**Базовый шорткод (все тикеры из файла):**
`[momentum_monthly]`

**С начальными параметрами:**
`[momentum_monthly lookback="12" holding="1" topn="15"]`

**Фильтрация по списку тикеров:**
`[momentum_monthly tickers="SBER,GAZP,LKOH,YNDX,GMKN"]`

Атрибут `tickers` позволяет размещать несколько скринеров на разных страницах
с разными наборами акций — данные загружаются из одного файла, расчёт ведётся
только по указанным тикерам. Тикеры из списка, которых нет в файле, молча игнорируются.

**Параметры шорткода:**

* `lookback="6"` — период расчёта momentum (недели)
* `holding="1"` — период удержания (недели)
* `topn="10"` — количество акций в портфеле
* `tickers="SBER,GAZP,..."` — ограничить расчёт указанными тикерами

**Блокировка параметров (запрет изменения пользователем):**

* `lock_lookback="1"` — период расчёта momentum
* `lock_holding="1"` — период удержания
* `lock_topn="1"` — количество акций
* `lock_dividends="1"` — тогл учёта дивидендов
* `lock_skip="1"` — слайдер Reversal Effect
* `lock_vol="1"` — фильтр волатильности
* `lock_riskadj="1"` — риск-корректированный momentum
* `lock_return="1"` — фильтр диапазона доходности

Пример с блокировкой:
`[momentum_monthly lookback="12" holding="1" topn="15" lock_lookback="1" lock_holding="1"]`

== Часто задаваемые вопросы ==

= Какой формат данных нужен? =

Excel (.xlsx) с листом «цены»:
- Первый столбец: Time (даты)
- Остальные столбцы: тикеры акций с ценами
- Каждая строка = один период (месяц)

= Как добавить дивиденды? =

Добавьте в Excel-файл лист «дивиденды» или «Дивид» в том же формате — тикеры в столбцах, суммы дивидендов по периодам.

= Как разместить два скринера с разными наборами акций? =

Используйте атрибут `tickers` на разных страницах:

Страница 1 (все акции): `[momentum_monthly]`
Страница 2 (выборка): `[momentum_monthly tickers="SBER,GAZP,LKOH,YNDX,GMKN"]`

= Что такое Reversal Effect (пропуск недель)? =

Исключает N последних недель из расчёта momentum, чтобы избежать краткосрочных разворотов. Классически пропускают 4 недели.

= Как работает риск-корректированный momentum? =

Вместо простой доходности использует отношение доходность/волатильность, что позволяет отдавать предпочтение акциям с более стабильным ростом.
